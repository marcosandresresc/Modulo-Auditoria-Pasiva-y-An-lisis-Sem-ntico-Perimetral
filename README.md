Módulo de Reconocimiento Web Pasivo y Análisis Adaptativo

Desarrollo de un script automatizado en Python (utilizando la librería HTTPX) que realiza una conexión inicial pasiva a un objetivo web para extraer y auditar de forma inteligente sus cabeceras de seguridad, cookies de sesión y firmas de software. A diferencia de un escáner básico que solo mira si las cosas existen, este motor procesa el texto interno de la respuesta para evaluar si las configuraciones realmente protegen al sitio o si tienen fallas lógicas que exponen información.
